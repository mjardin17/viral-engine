# Automation

All automation scripts and batch files for Viral Engine.

## Key Scripts
| Script | Purpose |
|--------|---------|
| LAUNCH_EMPIRE.bat | Start Empire OS server |
| START_EMPIRE_OS.bat | Start with auto port-kill |
| council_run.bat | Launch all 9 council bots |
| render_ep006.bat | Re-render GG EP006 (Pearl Harbor) |
| render_season3.bat | Render all 14 S3 episodes |
| COMMIT_SERVER.ps1 | Deploy + push to GitHub |
| EMPIRE_AUDIT.ps1 | Full system health audit |
| auto_render.py | Core: JSON → images → TTS → MP4 |
| patch_fallbacks.py | Fix broken/tiny images |
