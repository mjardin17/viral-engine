## What This Project Is
The Viral Engine Pipeline is a fully automated AI video production system. It takes episode topics and produces publish-ready MP4 documentary videos across three YouTube channels. It handles everything from script → images → narration → rendering → final video. It does NOT publish. Publishing is handled by the Social Machine (internal) and CrossPost (external).

## What This Project Is NOT
- Not a publishing system
- Not a social media manager
- Not an AI model
- Not an orchestrator

## Primary Output
Finished MP4 files in `renders/` directory, named `{CHANNEL}_{EP_ID}_final.mp4`

## Channels Served
| Channel | ID | Content Type | Episode Length | Scenes |
|---|---|---|---|---|
| Gods & Glory | GG | History documentary | ~18-20 min | 24 |
| Little Olympus | LO | Kids mythology cartoon | ~63 sec | 7 |
| Mech Legends | ML | Sci-fi action anime | ~90 sec | 10 |
