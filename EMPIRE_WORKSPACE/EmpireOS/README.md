# EmpireOS

The central nervous system. Node.js/TypeScript server running on port 3001.

## Start
```
LAUNCH_EMPIRE.bat
```

## Key URLs
- Health: http://localhost:3001/health
- Executive Briefing: http://localhost:3001/executive/
- Video Factory: http://localhost:3001/video-factory/
- Dashboard: EMPIRE_LIVE_DASHBOARD.html

## Modules
| Module | Route | Purpose |
|--------|-------|---------|
| executive | /executive/ | 10 AI workers, Master Queue, Daily Briefing |
| video-factory | /video-factory/ | 19 departments, 20-stage pipeline |
| knowledge-base | /knowledge-base/ | Searchable KB |
| media-engine | /media-engine/ | Asset generation |
| discovery | /discovery/ | Content discovery |
| benchmark-engine | /benchmark-engine/ | Model benchmarking |
| self-improvement | /self-improvement/ | Self-optimization |
